#  Inventory Management App

A simple, professional inventory management mobile app built with **Flutter**  perfect for small businesses to track products and stock levels.

##  What It Does

This app helps small businesses manage their inventory by:

- ** Dashboard**  See total products, inventory value, and low-stock alerts at a glance
- ** Product List**  Browse, search, and filter all inventory items
- ** Add/Edit Products**  Easy forms to add new items or update existing ones
- ** Delete Products**  Remove items with confirmation dialog
- ** Stock Alerts**  Visual warnings for low-stock and out-of-stock items

---

##  Key Features

| Feature | Description |
|---------|-------------|
| **Dashboard Overview** | Summary cards showing total items, stock value, low stock count |
| **Search & Filter** | Find products by name or category instantly |
| **Category Sorting** | Filter by Clothing, Footwear, Electronics, etc. |
| **Stock Status Badges** | Green (in stock), Yellow (low), Red (out of stock) |
| **Offline Storage** | Data saved locally using Hive database |
| **Sample Data** | Pre-loaded with 10 demo products to explore |

##  Quick Start

```bash
# Clone the repo
git clone https://github.com/EricElevencione/Inventory_Managment_App.git

# Navigate to project
cd Inventory_Managment_App

# Install dependencies
flutter pub get

# Run the app
flutter run
```
